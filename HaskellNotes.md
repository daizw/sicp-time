# 运算符 #

| **运算符** | **含义** | **示例** |
|:--------------|:-----------|:-----------|
| && | logical "and" | True && False |
| `|``|` | logical "or"  | Flase `|``|` True |
| /= | 不等于 | 2 /＝ 3 |
| ^ | (整数次)幂 | 3 ^ 2 |
| `*``*` | (浮点数次)幂 | 3 ^ 2.0 |
| ++ | 连接两个lists | `[3,1,3] ++ [3,7]` |
| : | 将一个元素插入列表头部 | `1:[2,3]` |
| % | 比例(表示分数) | 11 % 29 |
